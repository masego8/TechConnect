Install Dependencies ( Terminal Commands)

1. node -v (to check JavaScript version. if nothing download node.js)

2. npm install (install all dependecies for this project)

3. npm run dev (launch server)

Create '.env' file

find the url and anon key in supabase project

VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-supabase-public-anon-key
